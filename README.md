# Harsha's Cybersecurity Journey

I made this repo because I want to do something meaningful besides chasing unrealistic things in the cyber world like OSCP, at least for me right now.

I'm putting this out here so people like me know there's someone like them. You might look at this and not learn a single thing, and that's okay. At least you'll feel like okay, there's a person out there who's figuring it out and still doing it, not perfectly, but doing it.

For me personally, I want to document my journey. There's no scarcity of guidance in this field but I still feel like there's no clear path just big jargons and people telling you what to do without telling you how it actually feels to start from zero. So I'm starting at the very basic. I don't even know how I'm going to write notes yet.

The plan, loosely: THM → HTB → eJPT. If I fail or get stuck I'll go back to fundamentals without hesitation or fear of being judged. It's about trying, not dreaming. If not today, when.
